# project-maps

## visualize your project

![](https://s3.bmp.ovh/imgs/2021/12/975278670cc4d3eb.png)

## 1. install

```bash
yarn add project-maps
```

## 2. scripts

add script to package.json(scripts block)

```json
"scripts": {
 "maps": "project-maps"
},
```

- init config

```bash
yarn maps init
```

- generate html

```bash
yarn maps generate
```
